Embedded Key Value Stores

GRIP supports storing vertices and edges in a variety of key-value stores including:

  • Badger
  • BoltDB
  • LevelDB
  • RocksDB

Config:

# pick one of: badger, bolt, level, rocks
Database: badger

KVStorePath: grip.db
