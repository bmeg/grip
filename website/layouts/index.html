Download Documentation

A quick introduction

Start an Arachne server

$ arachne server --config arachne.yml

List all graphs

$ arachne list

Create a graph

$ arachne create example

Drop a graph

$ arachne drop example

Load data into a graph

$ arachne load example --edge edges.txt --vertex vertices.txt

Query a graph

$ arachne query example 'O.query().V().where(eq("_label", "users"))'

Get vertex/edge counts for a graph

$ arachne info example

Dump vertices/edges from a graph

$ arachne dump example --vertex
