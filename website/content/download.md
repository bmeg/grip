<h3>Build From Source</h3>

```shell
$ mkdir -p $GOPATH/src/github.com/bmeg
$ cd $GOPATH/src/github.com/bmeg
$ git clone https://github.com/bmeg/grip.git
$ cd grip
$ make
```
